Gatsby JS

Gatsby is a React-based, GraphQL powered, static site generator.

Created by Kyle Mathews

Pros and cons
  • Modern technology: React 16, GraphQL, Webpack, Babel 7 and more
  • Well documented, great tutorial, big community
  • Plugins and themes for more features
  • Official Starters for copy and customize
  • Pull data from any sources by GraphQL

Who are using GatsbyJS?
